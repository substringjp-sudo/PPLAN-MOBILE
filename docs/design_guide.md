# PPlan 디자인 가이드

## 1. 디자인 원칙

*   **직관성 (Intuitive):** 사용자가 별도의 학습 없이 기능을 쉽게 이해하고 사용할 수 있어야 합니다.
*   **일관성 (Consistent):** 앱 전체에 걸쳐 일관된 디자인 언어를 사용하여 통일성을 유지합니다.
*   **심미성 (Aesthetic):** 아름답고 감성적인 디자인을 통해 사용자의 만족도를 높입니다.
*   **명확성 (Clear):** 중요한 정보를 명확하게 전달하고 불필요한 시각적 요소를 최소화합니다.

## 2. 컬러 팔레트

| 역할 | 색상 코드 | 색상 | 설명 |
| :--- | :--- | :--- | :--- |
| **Primary** | `#6366F1` | Indigo | 앱의 핵심 컬러로, 버튼, 링크 등 주요 인터랙션 요소에 사용합니다. |
| **Secondary** | `#8B5CF6` | Violet | 보조 컬러로, 특정 UI 요소나 하이라이트에 사용합니다. |
| **Accent** | `#EC4899` | Pink | 강조 컬러로, 알림, 새로운 기능 등 주목이 필요한 부분에 사용합니다. |
| **Background** | `#F8FAFC` (Light) / `#1E293B` (Dark) | Slate | 앱의 배경색으로, 콘텐츠가 돋보이도록 합니다. |
| **Text** | `#111827` (Light) / `#F8FAFC` (Dark) | Gray | 일반 텍스트 컬러입니다. |

## 3. 타이포그래피 (Typography)

*   **기본 글꼴:** Google Fonts의 'Inter'를 사용하여 가독성을 높입니다.

| 스타일 | 크기 | 굵기 | 설명 |
| :--- | :--- | :--- | :--- |
| **Headline 1** | 24pt | Bold (700) | 화면 제목 등 가장 중요한 텍스트 |
| **Headline 2** | 20pt | Semi-bold (600) | 섹션 제목 |
| **Body 1** | 16pt | Regular (400) | 본문 텍스트 |
| **Body 2**| 14pt | Regular (400) | 보조적인 정보나 설명 |
| **Button** | 16pt | Medium (500) | 버튼 내 텍스트 |

## 4. 아이콘

*   **스타일:** Material Icons와 Cupertino Icons를 기본으로 사용하며, 필요시 커스텀 아이콘을 제작합니다.
*   **규칙:** 아이콘은 명확하고 단순한 형태로 제작하여 의미 전달력을 높입니다.

## 5. UI 컴포넌트

*   **버튼:** 채워진 버튼(Filled Button), 외곽선 버튼(Outlined Button), 텍스트 버튼(Text Button) 등을 상황에 맞게 사용합니다.
*   **카드:** 콘텐츠를 그룹화하여 보여줄 때 사용하며, 적절한 그림자와 여백을 가집니다.
*   **입력 필드:** 명확한 레이블과 플레이스홀더 텍스트를 제공합니다.
*   **네비게이션 바:** 앱의 주요 메뉴 간 이동을 담당하며, 일관된 디자인을 유지합니다.

*이 문서는 UI/UX 디자이너와의 협업을 통해 지속적으로 업데이트될 수 있습니다.*

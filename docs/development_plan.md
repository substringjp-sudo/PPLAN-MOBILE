# PPlan 개발 기획 문서

## 1. 프로젝트 개요

PPlan은 사용자의 일상과 여행 기록을 통합적으로 관리하고, 이를 타임라인과 스크랩북 형태로 시각화하여 제공하는 모바일 애플리케이션입니다. 개인의 소중한 순간들을 잊지 않고 언제든 다시 돌아볼 수 있는 디지털 앨범이자 개인 비서 역할을 수행하는 것을 목표로 합니다.

## 2. 목표 사용자 (Target Audience)

*   여행을 자주 다니며 기록을 남기고 싶어하는 여행자
*   일상의 소소한 순간들을 사진과 글로 기록하고 싶은 사람
*   여러 장소에서 찍은 사진들을 시간과 장소에 따라 정리하고 싶은 사람
*   자신만의 스토리를 담은 디지털 스크랩북을 만들고 싶은 사람

## 3. 주요 기능 정의

*   **타임라인:** 시간 순서에 따라 사진, 메모, 이동 경로 등을 자동으로 정리하여 보여줍니다.
*   **스크랩북:** 사용자가 직접 선택한 사진과 글을 모아 주제별 스크랩북을 만듭니다.
*   **여행 관리:** 여행 기간, 방문 장소, 관련 사진 등을 그룹화하여 관리합니다.
*   **스마트 추천:** 사용자의 기록을 바탕으로 관련성 높은 과거의 추억을 추천합니다. (고도화 기능)
*   **데이터 동기화:** Firebase를 통해 여러 기기에서 데이터를 안전하게 동기화합니다.

## 4. 기술 스택 및 개발 환경

*   **크로스 플랫폼 프레임워크:** Flutter
*   **백엔드:** Firebase (Authentication, Firestore, Storage)
*   **로컬 데이터베이스:** Isar
*   **상태 관리:** Riverpod
*   **라우팅:** go_router
*   **버전 관리:** Git & GitHub

## 5. 기대 효과

*   사용자는 흩어져 있던 사진과 기록을 하나의 앱에서 체계적으로 관리할 수 있습니다.
*   여행이나 특정 이벤트에 대한 추억을 쉽게 회상하고 공유할 수 있습니다.
*   자신만의 스토리를 담은 콘텐츠를 제작하며 새로운 즐거움을 느낄 수 있습니다.
